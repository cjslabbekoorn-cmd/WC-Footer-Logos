# Security Policy

## Supported Versions
We aim to support the latest stable WordPress, WooCommerce and Elementor versions.

## Reporting a Vulnerability
Please report security issues privately to: security@positie1.nl

## Notes
- This plugin can display SVG files if your WordPress installation allows SVG uploads.
  SVGs can be unsafe if not sanitized. For production use, only allow SVG uploads via a sanitizer
  (for example a reputable "Safe SVG" solution) and restrict upload permissions to trusted users.
